## 安装

```bash
$ npm install jgui
```

## 示例

```jsx
import { DatePicker } from 'jgui';
ReactDOM.render(<DatePicker />, mountNode);
```

引入样式：

```jsx
import 'jgui/dist/jgui.css';  // or 'antd/dist/jgui.scss'
```

按需加载可通过此写法 `import DatePicker from 'jgui/lib/date-picker'`
